Parameters to use in various fits:

f1 Analysis:

Procedure: change 'SetParameter' to 'FixParameter' for parameters you think are close.
Then read-off values for the other parameters.  Then change 'FixParameter' back to 'SetParameter',
set the values you read-off (change those to 'FixParameter'), and let the ones you had fixed float
(i.e. change the fixed ones to 'SetParameter').  Run the code and see if the fit is better, then collect the
revised values for the parameters you had fixed.

Finally, put the values you have collected into the code, set all parameters to 'SetParameter', and run the code.
Hopefully the fit will be better.  If not, repeat the procedure, or just try different parameters or ask 
for help.
